---

# **Purpose**

This guide helps determine which Consideration-stage blog type should be used for a content request.

At this stage, buyers have already:

```
✓ Recognized the problem

✓ Explored possible solutions

✓ Learned key concepts

✓ Understood available approaches
```

They are now asking:

```
How do we evaluate?

How do approaches compare?

How do we justify investment?

Are we ready to proceed?
```

---

# **Consideration Stage Definition**

## **Buyer Mindset**

The reader is actively evaluating.

They are attempting to reduce uncertainty before making a decision.

---

## **Primary Questions**

```
What should we evaluate?

How should we evaluate it?

Which approach fits us best?

What business value should we expect?

Are we prepared to move forward?
```

# **Decision Tree**

## **Step 1**

Is the content primarily about comparing approaches?

### **Examples**

```
Internal Mobility vs External Hiring

Rules-Based AI vs Agentic AI

Workforce Intelligence vs Workforce Reporting
```

If YES →

```
Comparison
```

If NO →

Continue.

---

## **Step 2**

Is the content primarily about business value, outcomes, or investment justification?

### **Examples**

```
The ROI Of Workforce Intelligence

Business Value Of Internal Mobility

Benefits Of Responsible AI
```

If YES →

```
ROI Narrative
```

If NO →

Continue.

---

## **Step 3**

Is the content focused on determining whether the organization is prepared?

### **Examples**

```
Are We Ready For Skills-Based Transformation?

Preparing For Workforce Intelligence

Readiness For AI Governance
```

If YES →

```
Implementation Readiness
```

If NO →

Continue.

---

## **Step 4**

Is the content focused on what criteria should be evaluated?

### **Examples**

```
What To Look For In Workforce Intelligence Platforms

Buyer's Guide To Internal Mobility Solutions

Evaluating Skills Intelligence Capabilities
```

If YES →

```
Buyer's Guide
```

If NO →

Continue.

---

## **Step 5**

Is the content focused on how to run an evaluation process?

### **Examples**

```
How To Evaluate Workforce Intelligence Platforms

Building An Evaluation Framework

Running A Solution Assessment
```

If YES →

```
Evaluation Guide
```

# **Blog Type Summaries**

---

## **Buyer's Guide**

### **Core Question**

```
What should we evaluate?
```

### **Purpose**

Help buyers understand evaluation criteria.

### **Reader Takeaway**

```
I know what capabilities and requirements matter.
```

### **Typical Outputs**

```
Buyer's Guide

What To Look For

Evaluation Criteria

Capability Assessment
```

---

## **Evaluation Guide**

### **Core Question**

```
How should we evaluate?
```

### **Purpose**

Help organizations build an evaluation process.

### **Reader Takeaway**

```
I know how to conduct a structured evaluation.
```

### **Typical Outputs**

```
Evaluation Framework

Assessment Process

Selection Methodology

Scoring Framework
```

---

## **Comparison**

### **Core Question**

```
How do approaches differ?
```

### **Purpose**

Help readers understand trade-offs.

### **Reader Takeaway**

```
I understand which approach fits our needs.
```

### **Typical Outputs**

```
X vs Y

Comparative Analysis

Approach Comparison

Strategy Comparison
```

---

## **ROI Narrative**

### **Core Question**

```
Why is this worth investing in?
```

### **Purpose**

Connect initiatives to business outcomes.

### **Reader Takeaway**

```
I understand the value and impact.
```

### **Typical Outputs**

```
Business Value

ROI Discussion

Investment Case

Outcome Analysis
```

---

## **Implementation Readiness**

### **Core Question**

```
Are we ready?
```

### **Purpose**

Assess organizational preparedness.

### **Reader Takeaway**

```
I know what needs to be in place before moving forward.
```

### **Typical Outputs**

```
Readiness Assessment

Preparation Guide

Transformation Readiness

Organizational Readiness
```

# **Common Classification Mistakes**

---

## **Mistake \#1**

### **Buyer's Guide vs Evaluation Guide**

Bad Classification:

```
How To Evaluate Workforce Intelligence Platforms
        ↓
Buyer's Guide
```

Correct:

```
How To Evaluate Workforce Intelligence Platforms
        ↓
Evaluation Guide
```

Reason:

The topic is about the process.

Not the criteria.

---

## **Mistake \#2**

```
        ↓
Buyer's Guide
```

```
        ↓
ROI Narrative
```

Reason:

The article is about outcomes.

# **Routing Priority**

When multiple templates seem possible:

```
1. Comparison

2. ROI Narrative

3. Implementation Readiness

4. Evaluation Guide

5. Buyer's Guide
```

---

# **Why This Priority Exists**

## **Comparison**

Usually easiest to identify.

Contains explicit alternatives.

---

## **ROI Narrative**

Usually driven by business outcomes.

Distinct from evaluation content.

---

## **Implementation Readiness**

Has a unique readiness mindset.

Rarely confused once identified.

---

## **Evaluation Guide**

Focused on process.

Often confused with Buyer's Guide.

---

## **Buyer's Guide**

Acts as the default Consideration structure when evaluation criteria are the primary topic.

# **Final Rule**

If the content helps the buyer answer:

```
What should we evaluate?
```

→ Buyer's Guide

```
How should we evaluate?
```

→ Evaluation Guide

```
Which approach fits us?
```

→ Comparison

```
Why invest?
```

This routing layer should sit directly beneath the **Master Blog Guide** and above the individual Consideration blog templates, serving as the decision engine for all Consideration-stage blog creation and automation.
